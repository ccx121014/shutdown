# 端口占用进程关闭工具

这是一个用于查找并关闭占用指定端口的进程的工具集，包含多个版本以满足不同的使用需求。

## 功能特性

- 支持查找占用指定端口的进程
- 显示进程详细信息（进程名、PID、启动时间、可执行文件路径）
- 支持关闭占用端口的进程
- 验证端口是否成功释放
- 提供多种使用方式：命令行交互版、命令行自动版、图形界面版

## 文件说明

| 文件名 | 功能描述 | 使用方式 |
|-------|--------|--------|
| `Close-PortProcess.ps1` | 命令行交互版，需要用户确认后关闭进程 | PowerShell: `.lose-PortProcess.ps1 -Port <端口号>` |
| `Close-PortProcess-Auto.ps1` | 命令行自动版，无需用户确认直接关闭进程 | PowerShell: `.lose-PortProcess-Auto.ps1 -Port <端口号>` |
| `Close-PortProcess-GUI-EN.ps1` | 图形界面版，提供可视化操作 | 直接双击运行或通过批处理文件启动 |
| `关闭端口进程.bat` | 批处理文件，用于启动图形界面版 | 直接双击运行 |

## 系统要求

- Windows 操作系统
- PowerShell 5.0 或更高版本
- 关闭某些系统进程可能需要管理员权限

## 使用方法

### 方法一：图形界面版（推荐）

1. 直接双击运行 `关闭端口进程.bat` 文件
2. 在弹出的窗口中输入要释放的端口号（默认值为8080）
3. 点击 "OK" 按钮
4. 查看找到的进程信息
5. 确认是否关闭这些进程
6. 查看操作结果

### 方法二：命令行交互版

1. 打开 PowerShell 终端
2. 导航到工具所在目录
3. 运行命令：`.lose-PortProcess.ps1 -Port <端口号>`
4. 查看找到的进程信息
5. 输入 "Y" 确认关闭进程
6. 查看操作结果

### 方法三：命令行自动版

1. 打开 PowerShell 终端
2. 导航到工具所在目录
3. 运行命令：`.lose-PortProcess-Auto.ps1 -Port <端口号>`
4. 工具会自动查找并关闭占用端口的进程
5. 查看操作结果

## 注意事项

1. **管理员权限**：关闭某些系统进程或其他用户的进程可能需要管理员权限。如果遇到权限不足的错误，请以管理员身份运行 PowerShell 或批处理文件。

2. **进程影响**：关闭进程可能会导致正在运行的程序被强制终止，请确保关闭的进程不会影响重要的系统功能或正在进行的工作。

3. **端口范围**：端口号必须在 1-65535 之间。

4. **兼容性**：该工具在 PowerShell 5.0 及以上版本中测试通过，可能不兼容较早版本的 PowerShell。

## 故障排除

1. **无法找到端口占用进程**：
   - 确认端口号输入正确
   - 确认该端口确实被占用
   - 尝试以管理员身份运行工具

2. **无法关闭进程**：
   - 以管理员身份运行工具
   - 确认进程不是系统关键进程
   - 检查进程是否被其他程序保护

3. **端口未成功释放**：
   - 尝试再次运行工具
   - 检查是否有其他进程重新占用了该端口
   - 重启计算机后再试
